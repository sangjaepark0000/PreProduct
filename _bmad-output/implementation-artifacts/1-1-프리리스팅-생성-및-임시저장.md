# Story 1.1: 프리리스팅 생성 및 임시저장

Status: done

## Story

As a 판매 사용자,
I want 첫 등록 시도를 안정적으로 시작하고 임시저장하고 싶다,
so that 중단 없이 다음 단계로 진행할 수 있다.

**Scope Tag:** Active MVP
**Story Type:** Feature
**FR Trace:** FR1, FR4, FR16, FR17

## Acceptance Criteria

1. **Given** 판매 사용자가 첫 등록 화면에 진입했을 때
   **When** 필수 필드(제목/카테고리/핵심 스펙 1개 이상/가격)를 입력해 임시저장을 요청하면
   **Then** 프리리스팅 초안이 저장된다
   **And** Story 1.2에서 즉시 검증 가능한 seed/fixture 기준이 문서화된다.
2. **Given** 부팅 또는 기본 검증 단계에서 실패가 발생했을 때
   **When** 사용자가 재시도를 수행하면
   **Then** 실패 원인과 복구 절차가 명확히 안내된다
   **And** 복구 후 동일 화면에서 임시저장을 다시 시도할 수 있다.

Coverage: FR1, FR4, FR16, FR17
